## ⚙️ Установка

1. Закройте игру, если она запущена;

2. Скачайте архив (zip/rar) со [страницы последнего релиза](https://github.com/sharewyd/aka_manto_ru_localization_and_bug_fixes/releases/latest);

3. В целях предосторожности (на случай нестабильной работы игры) скопируйте следующие файлы в отдельную папку на рабочем столе:

    3.1. Из папки "Aka Manto_Data" (обычно это: `C:\Program Files (x86)\Steam\steamapps\common\Aka Manto 赤マント\Aka Manto_Data\`);

    ```cmd
    level0
    level1
    level2
    level3
    level4
    resources.assets
    sharedassets1.assets
    ```

    3.2. Из папки "Managed" (обычно это: `C:\Program Files (x86)\Steam\steamapps\common\Aka Manto 赤マント\Aka Manto_Data\Managed\`);

   ```cmd
    Assembly-CSharp.dll
    ```
    
4. Распакуйте содержимое архива в папку с установленной игрой, подтверждая замену файлов (обычно это: `C:\Program Files (x86)\Steam\steamapps\common\Aka Manto 赤マント\Aka Manto_Data\`);

5. Запустите игру.

## ℹ️ Описание модификации

Данная модификация заменяет английский текст на русский и исправляет следующие ошибки:

Локация «Снаружи школы»:
- Во вкладке «ГРАФИКА» игрового меню исправлено отображение текста одной из настроек: второй пункт FULLSCREEN теперь правильно называется CRT EFFECT;
- Кнопка «ВЫЙТИ» в игровом меню теперь возвращает в главное меню.

Локация «Внутри школы»:
- После возврата в главное меню игрок теперь может снова начать новую игру;
- Во вкладке «УПРАВЛЕНИЕ» игрового меню исправлена ошибка, возникавшая при попытке назначить уже занятую клавишу.

Прочие исправления:
- Кнопка «ПО УМОЛЧАНИЮ» в игровом меню теперь корректно сбрасывает все настройки до исходных значений;
- Кнопка «ПРИМЕНИТЬ» в игровом меню теперь сохраняет все настройки не только для текущей локации, но и для последующих;
- Во вкладке «ГРАФИКА» игрового меню исправлено отображение раскрывающихся списков РАЗРЕШЕНИЕ и ВЕРТИКАЛЬНАЯ СИНХРОНИЗАЦИЯ;
- Различные мелкие исправления и улучшения.

К сожалению, в игре всё ещё осталось несколько незначительных ошибок, которые практически не влияют на игровой процесс, но встречаются такие ошибки крайне редко.
